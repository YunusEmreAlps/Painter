# Painter
**!!! Mouse is Active**

## Pen Move
-> Up    : 'Up'    or 'w' (lower case) <br/>
-> Down  : 'Down'  or 's' (lower case) <br/>
-> Right : 'Right' or 'd' (lower case) <br/>
-> Left  : 'Left'  or 'a' (lower case) <br/>

## Pen Color
-> Black : '1'  <br/>
-> White : '2'  <br/>
-> Red   : '3'  <br/>
-> Blue  : '4'  <br/>
-> Green : '5'  <br/>

## Basic shape
-> Square : '6'  <br/>
-> Circle : '7'  <br/>
-> Triangle : '8'  <br/>

## penup and pendown

-> penup : 'p' <br/>
-> pendown : 'o' <br/>
